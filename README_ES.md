## ¿Qué es Cortex-Nexus?

Cortex-Nexus es un framework validado empíricamente para inyectar estados afectivos estructurados en los prompts de LLMs para modular sistemáticamente la calidad del output. No es una librería ni un wrapper — es una metodología científica con evidencia experimental publicada.

### Las Tres Leyes Empíricas

1. **Ley de Masa Crítica** — Umbral mínimo de ~70B parámetros para efectos positivos.
2. **Ley de Especificidad de Dominio** — Maestría Técnica→código; Curiosidad Filosófica→razonamiento abierto.
3. **Ley de Saturación de Techo** — Modelos SOTA en dominios rígidos están cerca de una asíntota.
